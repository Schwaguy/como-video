Plugin to enable video management

== Installation ==
This section describes how to install the plugin and get it working.
e.g.
1. Upload `como-video.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode to embed videos
